# mern-todo-app
Implementation of todo app created with MERN stack


# Requirements:
Local MongoDB server on port 27017

Starting application:
1) Run following commands in both client and server directories:
```
npm install
```

2) Run in server directory

```
npm run app

```

# Used dependencies:
1) concurrently
2) express
3) mongoose
4) nodemon
5) react
6) react-redux
